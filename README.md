# 4Chan-Terminal-Viewer
A terminal based 4chan viewer using python.

## Requirements
* bs4
* requests
* html5lib

## Comands
* refresh : Refresh the thread
* only xxxxxxx : Show all messages (either sent or mentioned) of a certain users.
* back : go back to the previous screen
